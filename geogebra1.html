Geometry Constructions – Interactive GeoGebra Worksheet

Each construction task below includes an embedded GeoGebra applet. Students can follow the instructions, use the tools, and save their work.

1. Bisecting a Line Segment

Instructions: Draw segment AB → Construct arcs from A and B → Draw perpendicular bisector.

2. Constructing a Circle Through Three Points

Instructions: Plot points A, B, C → Construct perpendicular bisectors → Mark intersection → Draw circumcircle.

3. Angle Bisector

Instructions: Create angle ABC → Draw arc from B → Draw arcs from intersection points → Connect B to the intersection.

4. Perpendicular & Parallel Lines

Instructions: Try constructing a perpendicular from a point not on the line, and a parallel through a point above the line.

5. Triangle Centers (Incenter, Circumcenter, Centroid, Orthocenter)

Instructions: Construct a triangle → Add bisectors, medians, altitudes, or perpendicular bisectors → Observe how each center is formed.

6. Tangent Constructions

Instructions: Draw a circle and try constructing tangents from a point on or outside the circle.

This worksheet can be uploaded directly into a learning management system or used in class for hands-on exploration.
